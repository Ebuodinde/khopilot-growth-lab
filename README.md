# Khopilot Growth Lab

Khopilot pre-launch sosyal medya ve reklam kreatif workflow deposu.

Amaç: uygulama yayına çıkmadan önce Instagram/Threads/Facebook hesabında takipçi, merak, topluluk ve winner hook biriktirmek. Bu repo fikirleri, workflow'u, metrik şablonlarını ve Codex skill kurulumunu ayrı tutar.

## Yapı

- `skill/khopilot-growth-lab/`: Codex skill kaynağı.
- `workflows/`: Operasyon akışları.
- `ideas/`: İçerik fikir bankası.
- `templates/`: Brief, metrik ve haftalık review şablonları.
- `experiments/`: Yayın sonrası winner/loser notları.

## Codex Skill

Global kurulum hedefi:

```bash
~/.codex/skills/khopilot-growth-lab -> /Users/ebuodin/Developer/khopilot-growth-lab/skill/khopilot-growth-lab
```

Skill adı: `$khopilot-growth-lab`

Örnek kullanım:

```text
Use $khopilot-growth-lab to create today's pre-launch Instagram Reel brief for a natural motorcycle route hook.
```

## Ana Kural

App hazır değilken install satma. Hesap karakteri, rota kültürü, makara, tartışma, takip sebebi ve winner hook üret.
