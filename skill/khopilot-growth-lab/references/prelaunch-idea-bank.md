## First 7

Start with hooks 1, 2, 8, 13, 23, 12, and 24. They define account identity: road culture, humor, saveable routes, product reason, community debate, absurdity, and build-in-public.

## Clone Rule

Do not clone the exact sentence mechanically. Clone the tension:

- Same tension, new route.
- Same tension, other vehicle.
- Same tension, more absurd setup.
- Same tension, product-proof version.
